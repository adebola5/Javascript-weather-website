# Weather App (HTML, CSS, JavaScript)

## 📌 Description
A simple Weather App built with HTML, CSS, and JavaScript.

## 🚀 Features
- Search for any city
- Shows temperature, humidity, and weather description
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🙏 Acknowledgements
This project was built following a tutorial from [GreatStack](https://www.youtube.com/@GreatStack).  
I used the tutorial as a learning resource and then implemented the project myself.

## ▶️ How to Run
1. Clone or download the repository
2. Open `index.html` in your browser
